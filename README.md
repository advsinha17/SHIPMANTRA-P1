# SHIPMANTRA-P1

## Getting Started

To run a local copy of the application

### Installation

1. Clone the repository
    ```sh
    git clone https://github.com/advsinha17/SHIPMANTRA-P1
    ```
2. Install npm packages
    ```sh
    npm install 
    ```
3. Start the server
    ```sh
    node index.js 
    ```
